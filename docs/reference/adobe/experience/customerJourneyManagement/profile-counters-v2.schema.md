
# Profile Counters Extension Schema


## Profile Counters Extension Examples

```json
{
  "xdm:xid": "xid-92312748749128",
  "xdm:frequencyMap": {
    "counter_id": {
      "xdm:value": 100,
      "xdm:expiryDate": "2021-10-22T09:02:15.541Z"
    }
  }
}
```
