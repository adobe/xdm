## Point of Interest Details Examples

```json
{
  "xdm:POIID": "c7f4bf28-c8d9-4b89-a81f-2a8ef9367390",
  "xdm:name": "Acme Hotel Tokyo",
  "xdm:category": "Resorts",
  "xdm:type": "Hotel",
  "xdm:locatingType": "gps",
  "xdm:geoInteractionDetails": {
    "xdm:distanceToCenter": 100,
    "xdm:accuracy": 30
  }
}
```
