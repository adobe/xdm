## Site Search Examples

```json
{
  "xdm:filter": [
    {
      "xdm:attribute": "Device",
      "xdm:value": [
        "Mobile",
        "Phone"
      ],
      "xdm:isRange": false
    },
    {
      "xdm:attribute": "Price",
      "xdm:value": [
        "900",
        "1000"
      ],
      "xdm:isRange": true
    },
    {
      "xdm:attribute": "price",
      "xdm:value": [
        "800",
        "1000"
      ],
      "xdm:isRange": true
    }
  ],
  "xdm:sort": [
    {
      "xdm:attribute": "price",
      "xdm:order": "DESC"
    },
    {
      "xdm:attribute": "relevance",
      "xdm:order": "ASC"
    }
  ],
  "xdm:query": "Iphone 13"
}
```

```json
{
  "xdm:query": "Iphone 13",
  "xdm:suggestions": [
    "iphone 11",
    "iphone 12",
    "airpods"
  ],
  "xdm:numberOfResults": 35
}
```

## xdm:order
### Order

The order in which to return search results

`xdm:order`
* is optional
* type: `enum`
* defined in this schema

The value of this property **must** be equal to one of the [known values below](#xdmorder-known-values).

### xdm:order Known Values
| Value | Description |
|-------|-------------|
| `ASC` | Ascending order |
| `DESC` | Descending order |
| `RELEVANCE` | Sort items by relevance |
