## xdm:ID
### Ad ID

ID of the ad. Any integer and/or letter combination.

`xdm:ID`
* is optional
* type: `string`
* defined in this schema

### xdm:ID Type


`string`


All instances must conform to this regular expression 
(test examples [here](https://regexr.com/?expression=%5E.%7B1%2C%7D%24)):
```regex
^.{1,}$
```
